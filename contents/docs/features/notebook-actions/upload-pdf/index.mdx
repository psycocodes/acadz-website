# Uploading PDFs

AcadZ allows you to upload PDFs and convert their content into text using OCR. Here’s how to upload and convert a PDF into a transcript:

---

### Step 1: Open a Notebook
1. Navigate to the notebook where you want to store the PDF transcript.
2. Tap on the notebook to open it.

![Open Notebook](image-path)

---

### Step 2: Upload the PDF
1. Inside the notebook, tap the **Upload PDF** button.
2. You will be prompted to select the PDF file from your device.

![Upload PDF Button](image-path)

---

### Step 3: Select the PDF File
1. Browse your device’s storage to find the PDF file you want to upload.
2. Select the PDF file.

![Select PDF](image-path)

---

### Step 4: OCR Process
1. The app will automatically process the PDF using OCR technology to extract the text.
2. Wait for the OCR to finish processing the file.

![OCR Processing](image-path)

---

### Step 5: View and Edit the Transcript
1. Once the OCR process is complete, the app will display the extracted text as a transcript.
2. You can edit the transcript as necessary.

![OCR Result](image-path)

---

### Step 6: Save the Transcript
1. After editing the transcript, tap **Save** to store the content in the notebook.

You’ve successfully uploaded and converted a PDF into a transcript using AcadZ!
